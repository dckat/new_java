[직원 클래스]
 * 멤버변수로는 이름.주소.급여.주민번호를 가지고 있다.
  - 이름은 문자열이고 접근자는 public이다.
    --> public: 외부 클래스에서도 접근이 가능
  - 주소는 문자열이고 접근자는 default이다.
    --> default: 같은 패키지 내의 클래스끼리 접근이 가능
  - 급여는 정수형이고 접근자는 protect이다.
    --> protected: 부모-자식 클래스 관계에서만 접근이 가능
  - 주민번호는 정수형이고, 접근자는 private이다.
    --> private: 해당 클래스 내부에서만 접근이 가능
 * 멤버메소드는 toString을 가지고 있다.
  - toString() 메소드는 객체의 정보를 출력하기 위한 메소드로, 이를 재정의하여 사용하도록 한다.

[매니저 클래스]
 * 매니저 클래스는 직원 클래스로부터 상속을 받는 클래스이다.
 * 클래스에서 추가로 갖는 멤버변수로는 보너스 변수가 있다.
  - 보너스는 정수형이고, 접근자는 private이다.
 * 멤버 메소드는 test 메소드를 가지고 있다.
  - void 형태의 메소드로 단순히 출력용도로 사용한다.
